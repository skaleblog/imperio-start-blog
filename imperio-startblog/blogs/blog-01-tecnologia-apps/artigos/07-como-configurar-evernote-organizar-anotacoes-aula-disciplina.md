# Como configurar o Evernote para organizar anotações de aula por disciplina

Muitos estudantes têm o problema de acumular anotações em lugares diferentes — cadernos físicos, arquivos de texto soltos no celular, fotos de slides e documentos no Google Drive. Depois de algum tempo, encontrar a anotação certa antes de uma prova vira uma tarefa desgastante. O Evernote é um aplicativo projetado especificamente para centralizar e organizar esse tipo de conteúdo. Com sua estrutura de cadernos e tags, ele permite que você guarde anotações de aula por disciplina de forma organizada e fácil de recuperar. Neste artigo, você vai aprender a configurar o Evernote para organizar as anotações das suas disciplinas universitárias.

## Entendendo a estrutura do Evernote: cadernos, notas e tags

O Evernote organiza o conteúdo em três camadas:

- **Cadernos (Notebooks)**: funcionam como pastas principais. Você pode ter um caderno por disciplina ou por semestre.
- **Notas (Notes)**: são os documentos individuais dentro de cada caderno. Cada aula pode ser uma nota separada.
- **Tags (Etiquetas)**: permitem classificar notas com categorias cruzadas, independentemente do caderno onde estão. Por exemplo, a tag "prova" pode marcar notas de cadernos diferentes.

Essa estrutura é mais poderosa do que simplesmente jogar tudo em uma pasta, porque as tags permitem filtrar o conteúdo de formas que pastas tradicionais não permitem. Uma nota pode pertencer a um caderno e também ter múltiplas tags.

O plano gratuito do Evernote permite criar cadernos e notas com texto, imagens e outros conteúdos, sendo suficiente para a maioria dos estudantes.

## Como criar cadernos por disciplina e organizar as notas de aula

Após instalar o Evernote e criar uma conta, o primeiro passo é criar a estrutura de cadernos. No aplicativo (disponível para iOS e Android), toque em "Cadernos" no menu inferior e depois em "Novo caderno". Crie um caderno para cada disciplina que você está cursando.

Para organizar por semestre, você pode usar uma convenção de nomes como "2024-1 — Cálculo" ou "2024-1 — Filosofia". Isso facilita a localização dos cadernos corretos quando você tiver vários semestres acumulados.

Dentro de cada caderno, crie uma nota para cada aula. Use como título a data e o assunto principal — por exemplo, "15/03 — Derivadas e suas aplicações". Essa convenção garante que as notas fiquem em ordem cronológica e facilita localizar o conteúdo por assunto.

Ao criar a nota, use os cabeçalhos (H1, H2) disponíveis na barra de formatação para estruturar o conteúdo em seções: introdução do tema, conceitos principais, exemplos e dúvidas. Isso torna a nota muito mais fácil de revisar na véspera da prova.

## Como usar tags para criar categorias de revisão

As tags são uma das funcionalidades mais subutilizadas do Evernote por estudantes iniciantes. Para criar uma tag, abra uma nota, toque no campo "Adicionar tag" e digite o nome da tag. Ela é criada automaticamente na primeira vez que você a usa.

Crie tags funcionais como:
- **"revisar"**: para notas com conteúdo que você ainda não domina bem
- **"prova"**: para marcar assuntos que o professor indicou que cairão na prova
- **"dúvida"**: para partes da aula que ficaram confusas e precisam ser esclarecidas
- **"leitura"**: para notas que resumem textos obrigatórios

Para filtrar por tag, acesse o menu "Tags" no aplicativo. Ao tocar em "revisar", por exemplo, você vê todas as notas marcadas com essa tag em todos os cadernos — perfeito para criar uma lista de revisão antes das provas, independentemente da disciplina.

## Como usar o Evernote para capturar conteúdo além de texto

O Evernote permite adicionar diferentes tipos de conteúdo a uma nota. Para estudantes, os recursos mais úteis são:

**Fotos de quadro ou slides**: toque no ícone de câmera dentro da nota para fotografar diretamente. O Evernote aplica processamento de imagem que melhora a legibilidade de textos fotografados.

**Digitalização de documentos**: o Evernote tem um scanner integrado que reconhece automaticamente bordas e melhora a qualidade de papéis fotografados. Útil para digitalizar handouts ou exercícios entregues pelo professor.

**Anexos de documentos**: você pode anexar arquivos PDF, planilhas e apresentações diretamente às notas. Isso é ideal para guardar o material de aula junto com suas anotações do mesmo dia.

Tudo isso dentro de uma única nota que fica no caderno da disciplina correspondente, criando um registro completo de cada aula.

## Dica extra: use a função de pesquisa para encontrar qualquer informação rapidamente

Uma das maiores vantagens do Evernote é a sua busca interna. O aplicativo indexa todo o texto das suas notas — incluindo texto dentro de imagens — e permite buscas precisas.

Para encontrar qualquer informação, toque na lupa no topo do aplicativo e digite o termo que você procura. O Evernote mostra todas as notas que contêm aquela palavra, em qualquer caderno. Você pode refinar a busca por data, tag ou caderno específico usando os filtros avançados.

Isso significa que, na véspera da prova, você pode pesquisar "segunda lei de Newton" e encontrar instantaneamente todas as notas que mencionam aquele conceito, mesmo que você não lembre em qual aula foi explicado.

## Conclusão

O Evernote oferece uma estrutura robusta para quem quer organizar anotações de aula por disciplina de forma séria. Com cadernos por matéria, notas organizadas cronologicamente, tags para categorias de revisão e recursos de captura de fotos e documentos, você centraliza todo o seu material de estudo em um único lugar acessível de qualquer dispositivo. Configure sua estrutura de cadernos hoje, crie as tags que fazem sentido para sua rotina e comece a usar nas próximas aulas.
