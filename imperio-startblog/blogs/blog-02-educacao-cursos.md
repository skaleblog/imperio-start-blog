## Fase de Fundação

### Nicho de Aprovação — Técnica das 4 Camadas

| Camada | Termo |
|--------|-------|
| 1ª | Inglês |
| 2ª | Aprender inglês |
| 3ª | Aprender inglês em casa |
| 4ª | **Como aprender inglês assistindo séries na Netflix para adultos acima de 40 anos** |

**Validação do nicho:**
- Pesquise no Google: `como aprender inglês assistindo séries na Netflix para adultos acima de 40 anos`
- Ferramentas → verificar quantidade de resultados
- Meta: **menos de 200.000 resultados**
- Se passar: nicho aprovado

**Por que este nicho funciona:**
- Completamente fora do YMYL (educação, não saúde ou finanças)
- Público específico (40+) reduz concorrência drasticamente
- IA escreve dicas de aprendizado com qualidade alta
- Conteúdo perene — o problema de "aprender inglês" é eterno
- Empresas de cursos de inglês pagam muito caro no AdSense

## Checklist de Lançamento

- [ ] Validar nicho no Google (< 200.000 resultados)
- [ ] Escolher nome do blog e domínio
- [ ] Contratar hospedagem + instalar WordPress
- [ ] Instalar tema Astra/White ou Neve
- [ ] Criar logo e favicon com IA
- [ ] Instalar plugins: Jetpack, Instant Indexing, Contact Form
- [ ] Criar 6 páginas obrigatórias (Privacidade, Termos, Sobre, Cookies, Comentários, Contato)
- [ ] Criar bio + foto do redator
- [ ] Configurar sitemap + robots.txt + Search Console
- [ ] Escrever e publicar 30 artigos de base (IA + revisão)
- [ ] Inserir imagem em destaque em cada artigo
- [ ] Verificar indexação (Search Console + Instant Indexing)
- [ ] Executar checklist final do método antes de enviar
- [ ] Fazer inscrição no Google AdSense
- [ ] Enviar blog para aprovação
- [ ] Criar campanha PlugRush após envio
